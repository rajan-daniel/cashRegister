## 💵 payflo Cash Register Application (HTML, CSS & JavaScript Project)

### 📌 Preview  
A clean and interactive web app that simulates a real cash register, allowing users to enter a payment amount and instantly calculate the correct change due.

---

### 📖 About the Project  
**payflo** is a functional cash register application built using JavaScript, focusing on real-world transaction logic and dynamic user interaction. The app takes a user’s payment input, compares it against the total price, and calculates the correct change based on the available cash in the drawer.

It handles multiple scenarios such as insufficient funds, exact payment, or successfully returning change with the correct denominations. The register dynamically updates to reflect the remaining cash after each transaction, creating a more realistic and interactive experience.

The interface features a clean, modern design, and includes a custom banner I created using Paint.NET to give the application a more polished and branded look.

---

### 🚀 What I Learned  
- Strengthening my understanding of **JavaScript algorithms** and real-world logic  
- Working with **arrays and nested data structures** to represent cash denominations  
- Implementing **loops and conditionals** for calculating change efficiently  
- Handling **floating point precision** in currency calculations  
- Managing **application state** (cash in drawer) and updating it dynamically  
- Using **event listeners** to process transactions  
- Dynamically updating the **DOM** to reflect results and register status
